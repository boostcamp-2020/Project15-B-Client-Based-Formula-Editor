# 클라이언트 기반 수식 편집기

## 프로젝트 소개
사용자가 필요한 수식을 어느 브라우저든 어느 기기에서든 작성할 수 있는 반응형 웹 편집기  

- [배포 주소](https://kr.object.ncloudstorage.com/client-based-formula-editor-b/index.html)  
- 기간 2020.11.16 ~ 2020.12.18

## 실행방법 및 테스트 방법

  실행

  ```bash
  $ npm install
  $ npm start
  ```
  테스트
  ```bash
  $ npm test
  $ npm test -- --coverage
  ```
  린트
  ```bash
  $ npm run lint
  ```
